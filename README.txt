# Juego de Memoria con Turtle

Este es un programa de Python que utiliza los módulos `turtle`, `random`, `string` y `freegames` para crear un juego interactivo de memoria. 
En este juego, se generan 64 tarjetas ocultas en la pantalla y el jugador debe hacer clic en las tarjetas para revelarlas y encontrar pares de letras.

## Funciones

El programa consta de varias funciones:

- `square(x, y)`: Dibuja un cuadrado blanco con borde negro en las coordenadas (x, y).
- `index(x, y)`: Convierte las coordenadas (x, y) en un índice de tarjetas.
- `xy(count)`: Convierte el recuento de tarjetas en coordenadas (x, y).
- `tap(x, y)`: Actualiza la marca y las tarjetas ocultas en función del clic del usuario.
- `draw()`: Dibuja la imagen y las tarjetas en la pantalla.

## Uso

Para usar este programa, simplemente ejecútalo en un entorno de Python que tenga los módulos `turtle`, `random`, `string` y `freegames` instalados. 
Puedes interactuar con el programa haciendo clic en las tarjetas para revelarlas y encontrar pares de letras.

¡Diviértete jugando con Turtle!
